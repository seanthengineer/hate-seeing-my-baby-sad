# Vintage Love Story Website

A romantic, static website template for GitHub Pages.

## How to Edit

### 1. Text
Open `index.html` in any text editor (like Notepad or TextEdit). Look for comments that say ``. You can change the title, the letter, the photo captions, and the "100 Things" list.

### 2. Images
1. Go to the `assets/images/` folder.
2. Delete the placeholders (if any).
3. Add your own photos. 
4. Rename your photos to `photo1.jpg`, `photo2.jpg`, etc., OR update the `src="..."` code in `index.html` to match your new file names.

### 3. Audio
1. Go to the `assets/audio/` folder.
2. Add your MP3 file.
3. Rename it to `song.mp3`.

## How to Publish on GitHub

1. Create a new repository on GitHub named `my-love-website`.
2. Upload all these files (`index.html`, `style.css`, `script.js`, and the `assets` folder) to the repository.
3. Go to **Settings** > **Pages**.
4. Under **Source**, select `main` branch.
5. Wait 1-2 minutes, and GitHub will give you a link (e.g., `yourusername.github.io/my-love-website`).
